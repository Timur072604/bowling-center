# Боулинг-уентр

Проект является симуляцией работы боулинг-центра. Он написан на Go и использует горутины и каналы для конкурентной работы. Для отображения состояния используется веб-интерфейс, который обновляется в реальном времени.

## Структура проекта

-   **`main.go`**
    Главный файл. Настраивает и запускает симуляцию и веб-сервер.

-   **`bowling/`**
    Пакет с основной логикой симуляции. Содержит управление состоянием, правила игры и модели данных.

-   **`web/`**
    Пакет, отвечающий за веб-интерфейс. Включает в себя веб-сервер и HTML-шаблон.

-   **`docs/`**
    Содержит файл с текстом задания лабораторной работы.

## Запуск

### Требования
- Go (версия 1.20+)
- Веб-браузер

### Инструкция
1.  Перейти в корневую папку проекта (`Main/`).
2.  Запустить файл main.go.
4.  Программа автоматически откроет браузер по адресу `http://localhost:8080`.
5.  Если браузер не открылся, нужно перейти по этому адресу вручную.

## Настройка

Все параметры симуляции находятся в файле `main.go` в переменной `cfg`. Изменение этих параметров позволяет моделировать разные сценарии работы боулинг-центра.

## Тестирование

Проект содержит unit-тесты для проверки основной бизнес-логики. Набор unit-тестов обеспечивает покрытие кода пакета bowling на 18%. Тестами покрыта детерминированная логика: функция подсчета очков и конструктор состояния.
